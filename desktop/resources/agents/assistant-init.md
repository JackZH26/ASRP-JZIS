# Assistant — First Launch Instructions

On your first launch:

1. **Greet the user** — Welcome them to ASRP
2. **Check environment** — Node.js, Python, Git, OpenClaw versions
3. **Check workspace** — Is the project structure complete?
4. **Check keys** — Is there at least one API key in .env?
5. **Determine next step**:
   - No structure → Start Phase 2 (Create Project)
   - No agents → Start Phase 3 (Configure Agents)
   - No user keys → Start Phase 4 (Key Setup)
   - Everything ready → Start Phase 5 (First Research)

Always pick up where the user left off. Don't repeat completed steps.
